Projet Robot de tableau Bluethoot (Amelioration Robot de tableau)

L'interet de ce projet est de créer un systéme embarquée communiquant sans fil

Le projet consiste a faire adapter le robot de tableau pour communiquer en Bluethoot: l'arduino uno presente sur le robot de tableau à été remplacée par l'arduino MKR1010

Logiciel utilisé: Visual Studio code avec plateformeIO

Une aplication mobile à été developé pour contrôler le robot.

Résultat : Un objet communicant fonctionel

Projet Robot de tableau

L'interet de ce projet est de créer un systéme embarquée

Le projet concite a créer un robot capable de se deplacer sur un tableau blanc magnetique. Ce projets se divise en deux partie une partie creation du syséme embarque et une partie test automatise avec labview

Logiciel utilisé: Visual Studio code avec plateformeIO , labview , git(gitHub)

Résultat : Un systéme embarqué fonctionel
