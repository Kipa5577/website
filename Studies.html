Projets

Les projets ont été réalisés en groupe et sont présentés du plus récent au moins récent.

Robot de tableau Bluetooth (Amélioration du robot de tableau)

Description : Ce projet visait à créer un système embarqué communiquant sans fil. Il consistait à adapter un robot de tableau pour une communication Bluetooth en remplaçant l'Arduino Uno par un Arduino MKR1010.

Logiciels utilisés : Visual Studio Code avec PlatformIO

Résultat : Un objet communicant fonctionnel

Une application mobile a été développée pour contrôler le robot.

Robot de tableau

Description : L'objectif était de concevoir un système embarqué capable de se déplacer sur un tableau blanc magnétique. Le projet comprenait deux volets : la création du système embarqué et des tests automatisés avec LabVIEW.

Logiciels utilisés : Visual Studio Code avec PlatformIO, LabVIEW, Git (GitHub)

Résultat : Un système embarqué fonctionnel

Logiciel de supervision de portique de remontée mécanique

Description : Développement d'un logiciel de supervision pour monitorer et contrôler un portique de remontée mécanique, assurant une gestion efficace et sécurisée des opérations.

Logiciels utilisés : C#

Résultat : Un logiciel fonctionnel de supervision

Électrocardiogramme avec une pince

Description : Conception d'un système d'électrocardiogramme utilisant une pince pour capter les signaux électriques du cœur, avec simulation et analyse des résultats.

Logiciels utilisés : LTSpice

Résultat : Un prototype simulé fonctionnel

Jeu vidéo

Description : Réalisé en groupe de quatre, ce projet consistait à développer un jeu vidéo entièrement codé sous Linux. La coordination a été assurée via Git pour gérer le travail collaboratif.

Logiciels utilisés : Linux, Git

Résultat : Un jeu vidéo fonctionnel

Électrocardiogramme avec des électrodes

Description : Conception d'un système d'électrocardiogramme utilisant des électrodes pour mesurer les signaux cardiaques, avec simulation et validation des résultats.

Logiciels utilisés : LTSpice

Résultat : Un prototype simulé fonctionnel
