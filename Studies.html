Les projets étaient réalisés en groupe. Les projets sont ordonnés du plus récent au moins récent

Projet Robot de tableau Bluetooth (Amélioration Robot de tableau)

Description: L'intérêt de ce projet est de créer un système embarqué communiquant sans fil. Le projet consiste à adapter le robot de tableau pour communiquer en Bluetooth : l'Arduino Uno présent sur le robot de tableau a été remplacé par l'Arduino MKR1010.

Logiciel utilisé: Visual Studio Code avec PlatformIO

Résultat: Un objet communicant fonctionnel

Une application mobile a été développée pour contrôler le robot.

Projet Robot de tableau

Description: L'intérêt de ce projet est de créer un système embarqué. Le projet consiste à créer un robot capable de se déplacer sur un tableau blanc magnétique. Ce projet se divise en deux parties : une partie création du système embarqué et une partie test automatisé avec LabVIEW.

Logiciel utilisé: Visual Studio Code avec PlatformIO, LabVIEW, Git (GitHub)

Résultat: Un système embarqué fonctionnel
