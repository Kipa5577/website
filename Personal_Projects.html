Projet Site Internet

Objectif
Créer ce site internet pour présenter mes projets et compétences.
Approche
J'ai codé ce site à partir de rien, en utilisant HTML, CSS et JavaScript.
Connaissances acquises
Programmation en HTML/CSS/JavaScript, hébergement de site internet avec Apache2.
Logiciels utilisés
Visual Studio Code, Linux.
Résultat
Ce site internet fonctionnel.
Matériel
Raspberry Pi.
